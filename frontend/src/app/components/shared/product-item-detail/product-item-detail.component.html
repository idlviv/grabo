TITLE GRABOFLEX START

MAIN TEXT Экономичное решение для небольших мультифункциональных спортзалов, где важно иметь долговечную и безопасную площадку для игр. Плотный вспененный слой на обратной стороне обеспечивает соответствующий комфорт ходьбы. Рекомендуем для покрытия пола небольших спортивных помещений, а также и помещений для игр в детских садах.

ADDITIONAL SECTOR1 Паркетная система Grabo StrongAir Elite, специально разработанная для больших нагрузок, предназначена для мультифункциональных спортзалов. Верхняя часть системы представляет собой трёхслойный спортивный паркет толщиной 22 мм, со стабильным средним слоем и износостойким спортивным лаком. Благодаря своему отличному энергопоглащающему основанию, система эффективно защищает суставы спортсмена, максимально способствует достижению выдающихся спортивных результатов.
